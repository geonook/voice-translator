Voice Translator - Privacy Policy

Last updated: January 2025

Key Point: Voice Translator processes your voice locally and only sends translated text to OpenAI using your personal API key. We do not store or access your data.

Information We Collect

Data Stored Locally

  • API Keys: Your OpenAI API key is stored securely in your browser
  • Settings: Language preferences and configuration settings
  • Usage Statistics: Translation count (stored locally only)

Data Processed

  • Voice Input: Processed locally by your browser's Web Speech API
  • Text for Translation: Sent to OpenAI API using your API key
  • Translation Results: Cached locally for performance

How We Use Your Information

Local Processing

  • Voice recognition handled entirely by your browser
  • Settings stored locally on your device
  • No voice data transmitted to our servers

Third-Party Services

  • OpenAI API: Text sent for translation using your API key
  • We do not store or access your API communications

Data Storage and Security

  • All personal data stored locally in your browser
  • Secure Chrome storage APIs used
  • HTTPS encryption for all communications
  • No data synchronized across devices unless you enable Chrome sync

Data Sharing

We do not share your personal information with third parties, except:

  • Text sent to OpenAI for translation (as required for the service)
  • When required by law

Your Rights and Choices

  • Delete all data by removing the extension
  • Clear cached translations through extension settings
  • Modify or remove your API key at any time
  • Disable subtitle display
  • Uninstall to stop all data processing

Third-Party Services

OpenAI

  • We use OpenAI's API for translation services
  • Review OpenAI's Privacy Policy
  • Your API usage subject to OpenAI's terms

Google Chrome

  • Extension uses Chrome's Web Speech API
  • Review Google's Privacy Policy

Data Retention

  • Local Data: Until you uninstall the extension
  • Cached Translations: Cleared when extension restarts
  • Settings: Until manually changed or removed

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this Privacy Policy from time to time. Material changes will be reflected by updating the "Last updated" date.

Contact Us

If you have questions about this Privacy Policy, please contact us through our GitHub repository.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
Note: This extension operates primarily on your device. The only external data transmission is text sent to OpenAI for translation using your own API key.
